PersistentObject Class Reference

Detailed Description

PersistentObject is the base class of all persistent objects.

It mainly implements an unique identifier for each instance (ObjectId), tracking of the persistent state, methods for setting and getting values as well as callback methods for lifecycle events.

initialize ( array  $data)

Initialize the object with a set of data.

This method does not validate, does not change the object's state and does not call any listeners. Any existing data will be overwritten. The data will also be used as base line for tracking changes.

Parameters
